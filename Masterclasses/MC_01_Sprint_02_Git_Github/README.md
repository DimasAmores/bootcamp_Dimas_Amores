### 📄 Git & GitHub Masterclass

**¿Qué aprenderás?**

* Flujo fundamental: staging area, commits y HEAD
* Historial, diffs y cómo deshacer cambios
* Ramas: creación, cambio y buenas prácticas de nombrado
* Fusión de ramas (merge) y resolución de conflictos
* Trabajo con repositorios remotos (push, pull, fetch)
* Colaboración en GitHub: Pull Requests, forks e Issues
* Flujo de trabajo profesional (Git Flow) y Conventional Commits
* Ejercicios prácticos integradores
